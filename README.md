# Shubham Bhardwaj — Portfolio

Built with React + Vite + Framer Motion.

## Get Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm run build
# Push to GitHub → Connect to Vercel → Auto-deploy
```

## Stack
- React 18
- Vite
- Framer Motion (animations)
- React-Leaflet (contact map)
- SASS (styles)

## Sections
1. Loader — Multilingual curtain reveal
2. Hero — Full-screen with oversized typography
3. About — Animated text lines + stats
4. Skills — Scroll-triggered progress bars
5. Projects — List with hover states
6. Experience — Self-drawing timeline
7. YouTube — The Rebel Diplomat section
8. Contact — Bangalore map + contact form
